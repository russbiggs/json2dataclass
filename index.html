JSON2dataclass converts JSON to Python dataclass definition(s). Paste or type a valid JSON string on the left in the JSON input and JSON2dataclass creates Python dataclass definition(s) on the right. JSON2dataclass makes some assumptions about your JSON input and may assume object definitions incorrectly. Heavily inspired by the excellent JSON-to-go.
